# ChatApp

## 📌 Giới thiệu  
ChatApp là một ứng dụng trò chuyện thời gian thực được xây dựng với React.js cho frontend và Spring Boot cho backend. Ứng dụng hỗ trợ nhắn tin cá nhân, nhóm, thông báo trực tiếp, và xác thực người dùng an toàn.

## 🚀 Công nghệ sử dụng  
- **Frontend:** React.js
- **Backend:** Spring Boot, WebSocket  
- **Database:** MySQL  
- **Xác thực:** JWT Authentication  
- **Realtime Communication:** WebSocket  

## 🔥 Các tính năng chính  
### 🧑‍💻 Người dùng  
✅ Đăng ký, đăng nhập, quên mật khẩu  
✅ Nhắn tin cá nhân và nhóm  
✅ Danh sách bạn bè & tìm kiếm người dùng  
✅ Gửi hình ảnh, emoji, file đính kèm  
✅ Thông báo tin nhắn thời gian thực  
✅ Trạng thái online/offline  

### 🎯 Quản trị viên

✅ Giám sát tin nhắn và báo cáo vi phạm
